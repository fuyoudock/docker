## <center>简单且易用的docker开发环境</center>
# 使用
### 源码使用
1. 拉取代码 `git clone https://github.com/fuyoudock/docker.git`
2. 进入代码目录 `cd docker`
3. 复制环境配置文件 `copy .env.example .env`
4. 根据需要修改.env文件中的配置
5. 根据需要启动容器 `docker-compose up -d`
